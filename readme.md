# Proyecto Final: Juego RPG - Programación 3

Este proyecto es un juego de rol (RPG) desarrollado en **Java** como trabajo final para la materia Programación 3. Se basa fundamentalmente en los conceptos de **Programación Orientada a Objetos (POO)**.

## Características Principales
*   **Razas**: Humanos, Elfos y Orcos.
*   **Modos de Juego**: Creación manual de personajes o generación aleatoria.
*   **Sistema de Batalla**: Enfrentamientos por turnos mostrados en consola.
*   **Recompensas y Persistencia**: Sistema de buffs para ganadores y guardado de historial de partidas.
*   **Diseño**: Basado en diagrama UML.

## Cómo ejecutarlo
1.  Clonar el repositorio.
2.  Abrir el proyecto en un IDE (IntelliJ, Eclipse, etc.).
3.  Ejecutar el archivo `Main.java`.
